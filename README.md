# LUNA
Frontend for accessing various aerospace services

To launch:
```
docker compose up --build
```

Currently serves on 5173:
- http://localhost:5173/


## Install Bun
`sudo apt-get install unzip`
`curl -fsSL https://bun.sh/install | bash`
`~/.bun/bin/bun install`
`sudo apt-get install build-essential`
